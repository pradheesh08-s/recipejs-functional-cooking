
# RecipeJS Functional Cooking

A modern, functional programming-inspired recipe companion app for the web. Easily browse, search, filter, and favorite recipes in a clean, responsive interface.

## Features

- **Functional Programming Approach:** Clean, modular JavaScript code using functional paradigms for maintainability and scalability.
- **Recipe Management:** Browse a curated list of recipes with details like time, difficulty, and description.
- **Search & Filter:** Instantly search recipes by title or description, filter by difficulty or quick recipes, and sort by name or time.
- **Favorites:** Mark recipes as favorites and view them in a dedicated filter.
- **Responsive UI:** Modern, accessible design with a dashboard layout and smooth user experience.

## Usage

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Use the controls to search, filter, sort, and favorite recipes.

## Tech Stack

- **HTML5 & CSS3:** Semantic markup and modern, responsive styles.
- **JavaScript (ES6+):** Functional programming patterns, modular code, and localStorage for favorites.

## Project Structure

- `index.html` – Main HTML file and app layout
- `style.css` – Styles for the dashboard and components
- `app.js` – Functional JavaScript powering the app logic
- `README.md` – Project documentation
